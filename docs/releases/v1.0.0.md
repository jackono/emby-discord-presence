# v1.0.0

## Highlights

- Dynamic Discord Rich Presence layouts with configurable templates.
- TMDB poster artwork and external metadata link support.
- Watching/listening activity typing and playback progress bar support.
- npm-based installer and service manager for macOS.
- Interactive setup and edit wizard with provider validation.

## Notes

- The npm CLI currently targets macOS.
- The setup wizard writes the runtime config to `~/.config/media-discord-presence/config.json`.
- The managed background service runs from `~/.local/share/media-discord-presence`.
